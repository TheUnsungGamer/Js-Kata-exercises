# 💪 JS Kata Exercises

Daily JavaScript training exercises — documenting the grind of learning to code from scratch.

This repo is where it all started. Before freeCodeCamp, before the projects — this is the very first place I wrote JavaScript and committed to figuring it out.

## What's In Here

Small, focused exercises covering:

- Variables, data types, and operators
- Functions and scope
- Arrays and objects
- Loops and conditionals
- DOM manipulation
- Algorithmic problem solving

Each file is a self-contained exercise or challenge. Some are from structured curricula, some are problems I invented for myself.

## Why This Repo Exists

I'm a self-taught developer. I don't have a CS degree or a bootcamp certificate. What I have is consistency — showing up every day and writing code until it clicks.

This repo is proof of that process, not just the polished end results.
